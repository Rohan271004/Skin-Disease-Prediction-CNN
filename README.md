# 🧠 Skin Disease Prediction using CNN

## 📌 Project Overview
This project is a **Deep Learning-based Skin Disease Prediction System** that uses a Convolutional Neural Network (CNN) to classify skin diseases from images.

It helps in early detection by analyzing skin images and predicting possible diseases.

---

## 🚀 Features
- 📸 Image-based skin disease prediction
- 🧠 CNN deep learning model
- ⚡ Fast and accurate predictions
- 🌐 Full-stack web application (Frontend + Backend)

---

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Django (Python)
- **Machine Learning:** TensorFlow / Keras
- **Others:** OpenCV, NumPy

## ▶️ How to Run the Project

### 1️⃣ Clone the repository

git clone https://github.com/Rohan271004/Skin-Disease-Prediction-CNN.git

cd Skin-Disease-Prediction-CNN


### 2️⃣ Backend Setup

cd backend
pip install -r requirements.txt
python manage.py runserver


### 3️⃣ Frontend Setup

cd frontend/cnn
npm install
npm run dev

## 📊 Model Details
- Model: Convolutional Neural Network (CNN)

- Input: Skin image
